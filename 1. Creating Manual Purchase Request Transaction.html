Creating Manual Purchase Request Transaction

Step-by-Step Instructions

Step 1:

  • From the Role Center, Click on Purchase and select Create Requisition.



Step 2:

  • Under General tab, update Description, Type, Category, Priority, Need date & Order Type.



  • Note: In the category dropdown, users can select Item (for items), Service (for Service) or Items and Service (for creating a mixed requisition of both item and service).


Step 3:

  • Under Lines, select all the items that needs to be added in the requisition.
    Fill in the Requested Qty , select the required cost centre and account code from the drop down.



  • Note: Account Description added with Account code in the Purchase Requisition down.
    The Account Description will get defaulted based on the account code selection.




Step 4:

  • Now a Dialog Box opens displaying the file Title, Description & Type (External/Internal).Click OK to attach files.



Step 5:

  • The attached files are available under FactBox.



Step 6:

  • Once all the items are included, click on Request Approval>> Send Approval Request.
