Creating Requisition worksheet-based purchase Request

Step-by-Step Instructions

Step 1:

  • From the Role Centre, under Actions and select Purchase>> Requisition Worksheet.



Step 2:

  • Click on the ( ) icon.



Step 3:

  • Select the requisition worksheet to be used and click on Ok.



    There are two ways through which items can be added in a worksheet.
    A.Stock Wanted
    B.Get Work Order Lines
    C.Catalog Items
    All three options are available under Replenish tab in the worksheet.



A. Stocks Wanted

Step 1:

  • In the requisition worksheet, click on Replenish>>Stock Wanted.



Step 2:

  • Filter out items in the Stocks Wanted screen.
    After selecting the search criteria’s, click on ( ) Click to Search icon to implement the search conditions.



Step 3:

  • Update the Required Qty.



Step 4:

  • Click on Add to Worksheet.



Step 5:

  • Selected items added in the worksheet.



Step 6:

  • Delete – For deleting the items from the filtered list of items.
    Add to Worksheet – Adding the selected items in the Requisition Worksheet.
    Create Item PR – For directly converting the selected items in the list to a PR.



Step 7:

  • Export to Excel – For exporting the selected items in an excel.
    Import from Excel – The users can extract the items in an excel, update the excel and then import the excel back into ShipPalm using this function.



B. Get Work Order Lines

Step 1:

  • In the requisition worksheet, click on Replenish>> Get Work Order Lines.



Step 2:

  • Filter and select the work order to import the items updated in the lines in the requisition worksheet.




Step 3:

  • Items from work order imported in the requisition worksheet.



Step 4:

  • For converting worksheets into a requisition, open the worksheet and click on Replenish>> Create Requisition.



Step 5:

  • Select Add to existing draft order (for adding the items in an existing draft requisition) or Create New (to create a new requisition for the selected items).



Step 6:

  • Requisition created with Draft status.



C. Catalog Items

Step 1:

  • In the requisition worksheet, click on Replenish>> Catalog Items.



Step 2:

  • Use the search criteria Price catalog names to filter the Catalog price names.
    By clicking on the three (…) dots, the catalog price names will be listed.



Step 3:

  • Select the Catalog Name and click OK.
    The items will be displayed and fill in the Required Quantity.Click Add to Worksheet.



Step 4:

  • Click on the Requisition work sheet and click OK to add items to work sheet.



Step 5:

  • Click on Replenish >> Create Requisition.
    Now the Purchase Requisition will be created.



Step 6:

  • Select Add to existing draft order (for adding the items in an existing draft requisition) or Create New (to create a new requisition for the selected items) and click OK.



Step 7:

  • Requisition created with Draft status.
    The generated Purchase Requisitions can be updated.
